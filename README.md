# 🌾 AGRO_CORE // Interface Cyber-Agro

Uma interface web conceitual e futurista que une a estética industrial/neon do **Cyberpunk** com o universo tecnológico da **Agricultura Avançada** (Agropunk).

## 🚀 Tecnologias Utilizadas
- **HTML5** Semântico
- **CSS3** Avançado (Custom Properties & Grid System)
- **Vanilla JavaScript** (Manipulação de DOM reativa)

## ♿ Funcionalidade de Acessibilidade
O projeto conta com controle dinâmico de escala de texto (`-A` e `+A`). Ao interagir com os controles, a propriedade global do documento altera a raiz (`html`), forçando o navegador a recalcular todas as proporções baseadas em unidades `rem` e `em` instantaneamente, garantindo legibilidade sem quebrar o design.

## 📁 Como Executar o Projeto
1. Baixe ou crie os arquivos `index.html`, `style.css` e `script.js` na mesma pasta.
2. Abra o arquivo `index.html` diretamente em qualquer navegador moderno.
